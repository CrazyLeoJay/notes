# Android SDK 构建开发

开发SDK主要是为了对第三方公司提供服务。

这里主要探讨[**手游SDK**](guide)相关的设计与开发，由于各个公司的情况不一样，这里将不涉及代码。指探讨设计方案。

使用到的开发环境和平台：

- java 8+
- kotlin
- Gradle
- Gradle plugins
- Android 5.0+ 
- Android SDK Tools (对应版本即可)
- Android Studio
- ApkTools

版本没有太多限制，可以根据实际情况版本调整实现

其中很多方案，其他类型的sdk也可以参考，比如：

- 组件化设计
- 插件设计
